# Document Control Procedure

**Versao:** 2026-06-16

## 1. Objetivo

Controlar criacao, revisao, aprovacao, versionamento e retencao de documentos do SGSI.

## 2. Documentos cobertos

- Politicas.
- Procedimentos.
- Registros de risco.
- Evidencias de auditoria.
- Relatorios de incidente.
- Matriz de controles.
- Inventarios.

## 3. Padrao minimo

Todo documento deve conter:

- titulo;
- versao ou data;
- dono/responsavel quando aplicavel;
- escopo;
- conteudo aprovado ou status de rascunho.

## 4. Revisao

- Politicas principais: anual.
- Registros de risco: trimestral ou apos incidente.
- Inventarios: trimestral.
- Evidencias: conforme evento.

## 5. Retencao

Registros de auditoria e compliance devem ser mantidos por pelo menos 3 anos, salvo exigencia legal, contratual ou operacional maior.
