## 2. Objetivo

Verificar se o SGSI esta documentado, implementado, mantido e produzindo evidencias suficientes para apoiar ISO/IEC 27001 readiness.

## 3. Areas auditadas

- Escopo do SGSI.
- Politica de seguranca.
- Avaliacao de riscos.
- SoA.
- Controle de acesso.
- Gestao de mudancas.
- Incidentes.
- Fornecedores.
- Backup e recuperacao.
- Evidencias tecnicas.
