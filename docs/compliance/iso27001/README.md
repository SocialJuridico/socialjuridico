# ISO/IEC 27001 Readiness Kit

Este diretorio contem a base inicial do SGSI do Social Juridico para preparacao rumo a ISO/IEC 27001:2022.

## Arquivos

- `ISMS_SCOPE.md`: escopo do SGSI.
- `INFORMATION_SECURITY_POLICY.md`: politica de seguranca da informacao.
- `RISK_ASSESSMENT_METHOD.md`: metodologia de avaliacao de riscos.
- `RISK_REGISTER.md`: registro inicial de riscos.
- `STATEMENT_OF_APPLICABILITY.md`: rascunho inicial de aplicabilidade.
- `ACCESS_CONTROL_POLICY.md`: politica de controle de acesso.
- `BACKUP_AND_RECOVERY_POLICY.md`: politica de backup e recuperacao.
- `SUPPLIER_SECURITY_POLICY.md`: politica de fornecedores.
- `INTERNAL_AUDIT_PLAN.md`: plano de auditoria interna.
- `DOCUMENT_CONTROL.md`: controle de documentos do SGSI.
- `ISMS_OPERATING_PROCEDURE.md`: rotina mensal, trimestral, semestral e anual do SGSI.
- `MANAGEMENT_REVIEW.md`: procedimento de revisao pela direcao.
- `CORRECTIVE_ACTION_REGISTER.md`: registro de nao conformidades e acoes corretivas.
- `INTERNAL_AUDIT_CHECKLIST_Q1.md`: checklist da primeira auditoria interna.
- `INTERNAL_AUDIT_REPORT_TEMPLATE.md`: modelo de relatorio de auditoria interna.
- `ISMS_IMPLEMENTATION_ROADMAP.md`: roteiro de implementacao ate certificacao.

## Importante

Estes documentos criam readiness e governanca inicial. Eles nao substituem auditoria de certificacao nem reproduzem a norma ISO/IEC 27001. A certificacao final deve ser conduzida por organismo certificador acreditado.
