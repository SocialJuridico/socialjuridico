## Critérios para revisão imediata

Um aviso de privacidade deve ser revisado imediatamente quando:
- Nova categoria de dados pessoais for coletada
- Novo fornecedor for adicionado como operador
- Base legal de um tratamento for alterada
- Nova transferência internacional for estabelecida
- Direito do titular afetado for modificado
- Legislação aplicável (LGPD, regulamentações ANPD) for alterada

---

## Conteúdo mínimo obrigatório (LGPD Art. 9)

Cada aviso de privacidade deve informar:
- [ ] Finalidade específica do tratamento
- [ ] Forma e duração do tratamento
- [ ] Identificação do controlador
- [ ] Informações de contato do controlador
- [ ] Informações sobre o uso compartilhado de dados
- [ ] Responsabilidades dos agentes de tratamento
- [ ] Direitos do titular (com referência à LGPD Art. 18)
